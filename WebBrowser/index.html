Web Browser

A web browser is, in essence, a device for downloading web pages (over the HTTPProtocol) and displaying (rendering) them in a form that is readable and understandable by users. Web pages are written in HTML (Hypertext Markup Language), which is so simple that you wouldn't have to be a CodeMonkey to understand it (caveat: if the HTML was written using Microsoft Frontpage, a part of the Microsoft Office suite, you do indeed need to be a CodeMonkey of the highest order to comprehend it. See also TheObfusticatedXContest).
