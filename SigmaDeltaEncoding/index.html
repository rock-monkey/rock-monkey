Sigma Delta Encoding

Umm.. simply, it records a starting point, then everything afterwards is just a note of what the difference is between this point and the next point. SACDs use this

Can't really be compared to PCM as it would give values as such:

1-bit resolution, and potentially infinite sampling rate

Not to be confused with Pulse Code Modulation
