Abandonware

Trading of software which is still technically under copyright, but which the owner has either disowned, or turns a blind eye to
  • Beneath A Steel Sky is now abandonware, as Revolution have released it into the public domain
  • Grand Theft Auto Vice City is NOT
  • DukeNukem 3D is NOT. Despite the source code being released to the public for free use, the graphics, sound, routines and maps still belong to 3D Realms. So don't steal. Buy it.
