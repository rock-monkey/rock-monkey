Windows Longhorn

Codename for an upcoming Operating System from Microsoft Corporation. Will be the new version of Microsoft Windows, and is due for release in 2006... no; 2007. Erm... 2008. Sure this time?
Key features include:
  • .NETification of everything
  • WinFS - Maybe. Unlikely it'll be finished in time
  • New version of Internet Explorer - maybe with real features this time!
  • 3D-accelerated WindowingSystem - Jon Atkinson: I think they're basically drawing everything onto an OpenGL/Direct3D surface, which means doing the processing on the screen can be offloaded onto the GPU. (Paul Mann: The downside is that it will require a 3D Accelerator card. Now, I do understand that everyone has one, but making one *required*? If I wanted an OS which wasted millions of CPU Cycles on pointless graphical novelties, I'd buy a Mac)
