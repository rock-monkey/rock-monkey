RockMonkey

Knightmare Wiki Game

Based on the hit 80's children's TV show Knightmare, this WikiGame is under development by Dan Huntley.
So, until you phase with us once more...

Extra Information

Got questions? Post them here!
  • The Adventurers Code: Please read it here. You'll play a lot better that way.
  • System Requirements: This WikiGame will not run in a satisfactory way under Microsoft Internet Explorer 6 or below (all released versions at time of writing). IT REALLY WILL NOT! You will need a web browser that correctly supports CSS and PNG images. Under Internet Explorer, the game will be pretty much unplayable. The WikiGame will be tested under Mozilla Firefox and Opera, which can be obtained from GetFirefox.com and Opera.com respectively.
  • Images Used: Images used in the game include stills from the series, as broadcast on ChallengeTV, some of which have been edited to, for example, airbrush out dungeoneers and props, which are then dynamically added later.
