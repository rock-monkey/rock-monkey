Extra Information

Got questions? Post them here!
  • The Adventurers Code: Please read it here. You'll play a lot better that way.
  • System Requirements: This WikiGame will not run in a satisfactory way under Microsoft InternetExplorer 6 or below (all released versions at time of writing). IT REALLY WILL NOT! You will need a web browser that correctly supports CSS and PNG images. Under InternetExplorer, the game will be pretty much unplayable. The WikiGame will be tested under MozillaFirefox and Opera, which can be obtained from GetFirefox.com and Opera.com respectively.
  • Images Used: Images used in the game include stills from the series, as broadcast on ChallengeTV, some of which have been edited to, for example, airbrush out dungeoneers and props, which are then dynamically added later.
