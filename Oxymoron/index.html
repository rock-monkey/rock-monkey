Oxymoron

An Oxymoron is a phrase that seems to contradict itself in it's own words. Common phrases which are also popular oxymorons include:
  • Government Organization
  • Alone Together
  • Personal Computer (Mmm - Paul)
  • Silent Scream
  • Living Dead
  • Same Difference
  • Taped Live
  • Plastic Glasses
  • Tight Slacks
  • Peace Force
  • Pretty Ugly
  • Head Butt
  • Working Vacation
  • Tax Return
  • Virtual Reality
  • Dodge Ram
  • Work Party
  • Jumbo Shrimp (Only if you're American. We tend to call them prawns - Paul)
  • Healthy Tan
  • Microsoft Works
  • FreedomOfSpeech

These, and many more, oxymorons can be found on OxymoronList.com.
