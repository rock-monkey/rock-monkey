Pulse Code Modulation

Umm... simply it looks at the incoming wave and approximates the voltage level to a known value (The number of values is referred to as the "resolution"). CDs use this, and have a range of values 65536 wide (16-bit), and the wave is looked at 44,100 times per second (44.1khz)

Not to be confused with Sigma Delta Encoding
