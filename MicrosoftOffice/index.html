MicrosoftOffice

An over-developed piece of software, which uses the 'document' metaphor for performing common office tasks such as work processing, keeping ledgers, sending mail and suchlike. It's only avaliable for MicrosoftWindows, which is no bad thing, because the rest of us can use the much more streamlined OpenOffice.
MicrosoftOffice consists of:
MicrosoftWord
MicrosoftExcel
MicrosoftFrontpage
MicrosoftAccess
MicrosoftOutlook
