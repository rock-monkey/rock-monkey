Rockipedia is an attempt to add some relatively valuable content to the RockMonkey Wiki. Despite being run by - and to some degree, about - a self-confessed RockMonkey, Andy Keohane, the site has not - until now - had any content of value to RockMonkeys.

Rockipedia will help to identify rocks for you. If you have a rock to hand, give it a go.
