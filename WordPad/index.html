WordPad

WordPad is a word processing application which is better that MicrosoftNotePad, but probably worse than MicrosoftWord. Only really useful because as of MicrosoftWindows 98, it was the only 'out of the box' application which could read UnixLineBreaks without BreakingHorribly.
Remember, if you use either for creating a ProperDocument, you're doing the WrongThing. For ProperDocuments you should use LaTeX.
Notepad isn't a word processing application. Ahem.
