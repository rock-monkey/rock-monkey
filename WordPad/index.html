Word Pad

Word Pad is a word processing application which is better that Microsoft Note Pad, but probably worse than Microsoft Word. Only really useful because as of Microsoft Windows 98, it was the only 'out of the box' application which could read Unix Line Breaks without Breaking Horribly.
Remember, if you use either for creating a ProperDocument, you're doing the Wrong Thing. For ProperDocuments you should use LaTeX.
