MicrosoftFrontpage

MicrosoftFrontpage is a web design program produced by MicrosoftCorporation and included in some editions of the MicrosoftOffice suite.
I hasten to add that it is not, as most web design programs are, a web page creator, but instead is an InternetExplorer page creator - as in, it makes pages that can be viewed in InternetExplorer, but not a lot else.
The HTML-code it produces is almost unusable for any purpose whatsoever.
