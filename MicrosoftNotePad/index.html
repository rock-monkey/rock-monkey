MicrosoftNotePad

"You're supposed to use the keyboard to put text on the screen. NOT a fountain pen."
MicrosoftNotePad is an ASCII text editor that comes packaged with MicrosoftWindows. It's only notable 'special' feature is an integrated search tool, which pretty much everything has anyway. It can handle only DOS-formatted line breaks.
It's useful only if you happen to be using a Windows box and the phone rings and you need something close to hand to take notes in. That's about it.
PaulMann would like to add it's one of the few well designed applications which comes with MicrosoftWindows. It has a tiny memory footprint, and it loads incredibly fast.. I didn't advocate it until the later versions where they removed the 64K file limit. Now my only complaint is that it doesn't properly support linebreaks which lack the CR-byte (Windows uses CR/LF, *nix and AmigaOS use LF only, MacOS uses CR only. From a purely technical point of view, you should probably be using just CR, since LF doesn't have a physical-meaning unless you're printing)
