HTMLLanguage

Hypertext Markup Language ("the language of the web") is an easy-to-learn document markup language used in the development of web pages. It is not necessarily a programming language as such, although recent enhancements - such as the ability to embed JavaScript code directly into the document, and the growth of server-side scripting languages such as PHP and ASP - have made it more useable as an application platform.
