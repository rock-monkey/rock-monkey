Microsoft Works

Microsoft Works is a cut-down version of Microsoft Office designed for home and small business usage by Microsoft Corporation. It consists of a single package which barely does what Word Pad can, but claims to be a reasonable replacement to Microsoft Word, Microsoft Excel, etc.

Basically, it blows chunks.
