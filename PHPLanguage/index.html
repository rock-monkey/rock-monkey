PHPLanguage

PHP is a ProgrammingLanguage designed for web server-side scripting. Indeed, this very web site is powered by PHP. It is fast, easy-to-learn (particularly coming from a Perl background), and vastly superior to ASP, it's major competitor.
Like most of it's competitors, PHP code is written directly into the HTML document (web page).
