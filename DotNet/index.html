DotNet

Microsoft .NET is a suite of inter-related APIs and ProgrammingLanguages, primarily for the MicrosoftWindows platform. Use of it is a very contentious issue amongst CodeMonkeys at the moment, with some geeks, like AlexMatthews, favouring it, while others, like DanHuntley and PaulMann, shunning it.
Microsoft intends to integrate .NET fully into the Windows OperatingSystem with WindowsLonghorn, due 2008.
The .NET ProgrammingLanguage suite includes:
  • MicrosoftVBDotNet
  • MicrosoftCSharpDotNet
  • MicrosoftJSharpDotNet
