Using The Chat Room

This is the RockMonkey Chat Room. If you're going to be hanging around on RockMonkey for hours on end, why not open this RockMonkey Wiki Page in a seperate Web Browser tab (or window, if you're using an inferior Web Browser), and keep in touch with other RockMonkey users.
Yes; it's IRC, which means that the usual commands work:
  • Use "/nick newname to change your name.
  • Use "/msg otherperson secret message" to send a message to another person.
  • Use "/quit Gotta go wash my socks" to quit, sending a logoff message to the room
  • If you want to connect with a 'traditional' IRC client, point it at a Freenode server (e.g. orwell.freenode.net), channel """#RockMonkey"""

    Igneous

    Igneous (or "Iggy", as he's known to his friends) is a robot who hangs around in the Chat Room. He's talkative, but a little stupid. Help him learn things by telling him about them (and ask him about them later!). He also helps collect the Chat Room Stats.
