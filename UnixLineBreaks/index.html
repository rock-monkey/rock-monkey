Unix Line Breaks

One fundamental thing that OperatingSystems can't agree on, is how to delimit the end of a line in a Text File.
Unix uses the ASCII character 32, while DOS (and it's derivatives, Microsoft Windows) uses 15.
Here is an easy way to convery a file with DOS linebreaks to a file with Unix linebreaks:
# tr -d '\15\32' < dos-format-file > unix-friendly-file
