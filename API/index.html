API

Application Programming Interface
An abstract way of talking to a computer using metaphors
For example, if you want to delete a file in DOS, you would type
del [name of file]
In Windows, you might drag it to a trashcan.
The idea is that you don't have to know how the hardware works to use it. This is why Operating Systems were invented. Then we invented OperatingEnvironments to talk to them, and now with the help of OSs like Mac Os, no-one really knows what's going on any more. It may be easy to use, but it does explain why your 3Ghz PC don't fundamentally move quicker than an 8Mhz AtariSt back in the early 90s
