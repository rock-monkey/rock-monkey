Bad Pages No Links

It is considerered bad wiki etiquette to create a Wiki Page which has no links to other WikiPages. At the very least, a Wiki Page should have a link back to the page that it was first referenced from. Ideally, a Wiki Page should provide links to other, related pages - use the search tool to find similar pages!
