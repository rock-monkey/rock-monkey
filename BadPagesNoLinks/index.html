BadPagesNoLinks

It is considerered bad wiki etiquette to create a WikiPage which has no links to other WikiPages. At the very least, a WikiPage should have a link back to the page that it was first referenced from. Ideally, a WikiPage should provide links to other, related pages - use the search tool to find similar pages!
