NickServ

NickServ is a bot on irc.freenode.net (the server where the RockMonkey ChatRoom is currently hosted). It's purpose is to keep tabs on who uses what NickNames on the server. You can stop people from stealing your nickname by registering it with NickServ.
Once you've registered your NickName, you authenticate to the bot and it will give you access to all the privelages you've been given (like OpsStatus in channels).

To register a NickName, you need to log into IRC, then do the following:
Give yourself the nickname you want (to make sure it's not already being used)
Register yourself with the command /msg nickserv register <your-password>
OPTIONAL: Then send /msg nickserv set hide email on
OPTIONAL: And finally /msg nickserv set email <your-email-address>
You can set more, such as an alternative nickname, if you want, but you don't have to
Whenever you connect to RockMonkey, send the command /msg nickserv identify <your-password> - most IRC clients allow you to define a list of commands to send when you first connect...
